Pour executer programme :
Veuillez ouvrir wamp ou tout autre programme de serveur local equivalent
Veuillez ouvrir un logiciel de BDD comme HeidiSQL en lançant le script "DahouëtSQLGymriin"
executer la requete ainsi chargée
veuillez lancer eclipse ou tout autre IDE Java
Importez le dossier Dahouët dans votre espace de travail
Lancez le main "Dahouët.java"

et c'est parti !!!
